Desenvolvi uma API utilizando a prática de DDD, com uma entidade Lead que contém uma lista de TaskItem, implementando operações de CRUD.

Para rodar o projeto no Visual Studio, basta executar o comando:

Update-Database

Esse comando cria o banco de dados localmente.

Após isso, a aplicação pode ser acessada pelo Swagger em:

https://localhost:7221/swagger/index.html
